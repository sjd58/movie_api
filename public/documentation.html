movie_api project documentation

Project description: The following is a simple project using node.js to build an API to send users information on their favorite movies.

Business Logic URL HTTP Method Request body data format Response body data format
Return a list of ALL movies to the user /movies GET none A JSON object hoding data about all the movies
Return data about a single movie by title to the user /movies/:title GET none A JSON object holding data about a single movie to the user
Return data about a genre (description) by name/title /movies/genre/:genreName GET none A JSON object holding data about a genre to the user by name/title
Return data about a director by name /movies/director/:directorName GET none A JSON object holding data about a director by name
Allow new users to register /users POST A JSON object holding data about the user to add, structured like:
{ "Username": "exampleUser1", "Password": "examplePassword1", "Email": "exampleemail@example.com" "Birthday": "01/01/2000" "favoriteMovies": [] } 		A JSON object holding data about the user that was added, including an ID:
{ "Username": "exampleUser1", "Password": "examplePassword1", "Email": "exampleemail@example.com" "Birthday": "01/01/2000" "favoriteMovies": [] }
Allow users to update their user info (username) /users/:id PUT A JSON object holding data about the user to add, structured like:
{ "name": "exampleUser1", } 		A JSON object holding data about the user's changed name structured like:
{ id: 1, "name": "exampleUser1", favoriteMovies: [] }
Return a list of ALL users to the user /users GET None A JSON object holding data about all users, structured like this example:
{ { "_id": "620fff65261f9b211379b2e0", "Username": "frodobaggins", "Password": "#theshire", "Email": "theshire@middleearth.com", "Birthday:" "04/01/1990", "FavoriteMovies": [] }, { "_id": "620fffb3261f9b211379b2e1", "Username": "bilbobaggins", "Password": "%bagend", "Email": "bagend@middleearath.com", "Birthday:" "09/12/1965", "FavoriteMovies": [] } }
Allow users to add a movie to their list of favorites /users/:id/:favoriteMovies PUT none A message indicating whether the movie was successfully added
Allow users to remove a movie from their list of favorites /users/:id/:favoriteMovies DELETE none A message indicating the favorite movie has been removed
Allow existing users to deregister /users/:id DELETE none A message indicating the user has been deleted
